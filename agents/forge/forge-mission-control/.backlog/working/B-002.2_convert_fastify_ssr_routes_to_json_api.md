## Goal

Convert all SSR routes to JSON API under /api/*. Add @fastify/static to serve dist/client/ in production. Add SPA fallback. Migrate execFileSync → async execFile (promisified) in all 3 providers. Keep events.ts SSE endpoint unchanged.

## Done When
- [ ] All GET routes return JSON at /api/product/*, /api/backlog/*, /api/agents/*
- [ ] POST routes: /api/backlog/item/:id/move, /api/agents/worker/:id/sync
- [ ] @fastify/static serves dist/client/ with SPA fallback to index.html
- [ ] All 3 providers use async execFile (no execFileSync)
- [ ] /events SSE endpoint unchanged
- [ ] /api/health returns JSON
- [ ] Does NOT delete old SSR render/ files yet (cleanup later)
