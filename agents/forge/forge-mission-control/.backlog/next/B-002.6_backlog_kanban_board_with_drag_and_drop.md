## Goal

Convert backlog kanban to KanbanBoard.tsx with @dnd-kit/core + @dnd-kit/sortable. 4 columns (Next/Working/Done/Archive). Draggable cards with kind badge + priority. Optimistic move via useMutation with rollback on error. BacklogItemPage for detail view (metadata, deps, markdown body). Wire to /api/backlog/* via React Query. SSE backlog_change invalidates query.

## Done When
- [ ] /backlog shows 4-column kanban board
- [ ] Cards are draggable between columns via @dnd-kit
- [ ] Drop triggers POST /api/backlog/item/:id/move with optimistic update
- [ ] Failed moves roll back to previous column
- [ ] /backlog/item/:id shows full item detail
- [ ] SSE backlog_change event invalidates backlog queries
- [ ] Cards show id, title, kind badge, priority badge
