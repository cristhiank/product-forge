# Agents Board

A shared blackboard for multi-agent coordination. See [SKILL.md](SKILL.md) for usage.

## Development

```bash
npm install            # Install dependencies
npm run build          # Compile TypeScript
npm test               # Run tests (vitest)
npm run bundle         # Bundle scripts/board.js
npm run publish:skill  # Build + publish to ~/.copilot/skills/agents-board/
```
